# 高德 搜索 POI 2.0

高德官方文档：https://lbs.amap.com/api/webservice/guide/api/newpoisearch



提供了可视化界面操作高德地点搜索服务2.0 Web API接口服务；

服务主要提供根据地区，以及 POI 搜索查询；

并支持批量导出结果功能；

并可以控制用户搜索次数；

支持批量添加高德 KEY；

个人认证：每天 100 次

企业认证：每天 1000 次
